
```py
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python3 get-pip.py
```

# Install Virtual Environment

```py
python3 -m venv venv
source venv/bin/activate
```

# Install Jupyter Notebook

```py
pip install notebook
```

# Run Jupyter Notebook

```py
jupyter notebook
```
